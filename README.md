# rigt-assignment

## Introduction

A simple admin dashboard build with Vue,Typescript,Pinia,I18n,zod and Vue router

## Project Setup

in your terminal type

```sh
git clone https://github.com/Ako-Mawlood/rigt-assignment
```

then

```sh
cd rigt-assignment
```

then

```sh
bun install
```

and

```sh
bun run dev
```
